# DataBaseTask
Задание на создание БД
Тут есть реализация на Python и реализация в PostgreSQL

## Реализация на Python
Можно делать такие команды:
Список действий:
1 - Загрузка базы из файла CSV\
2 - Сохранение базы в файл CSV\
3 - Вывод всех деталей на экран\
4 - Добавление новой детали\
5 - Поиск детали по названию\
6 - Удаление детали по названию\
7 - Агрегация данных по типу детали\
0 - Выход\

## Реализация на PostgreSQL
Создает таблицу details(\
	name VARCHAR(250) primary key,\
	type VARCHAR(250),\
	quantity INT);
